# 🏏 Virat Kohli Score Analysis Dashboard

An interactive Power BI dashboard analyzing Virat Kohli's 
batting performance across ODI and T20 internationals 
from 2008 to 2022.

## 📁 Dataset

- **Source:** [Virat Kohli Test/ODI/T20 Stats — Kaggle](https://www.kaggle.com/datasets/akashkotal/virat-kholi-testodit20-stat)
- **Coverage:** 2008 – 2022
- **Format:** CSV
- **Fields:** Year, Month, Quarter, Runs, Ground, Opponent, Date

---

## 📌 Dashboard Features

- 📅 **Year-wise filtering** — Interactive slicer from 2008 to 2022
- 🏟️ **Runs by Top 5 Grounds** — Bar chart of best performing venues
- 🌍 **Top 5 Performance by Grounds** — Table with opponent and location
- ⚔️ **Maximum Runs vs Opponent** — Best score against each team
- 📈 **Quarterly & Monthly Breakdown** — Matrix view by year/quarter/month
